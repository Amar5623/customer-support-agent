# Leafy Return & Refund Policy

## Overview

We want you to love what you ordered. If something isn't right, we make returns
straightforward — no restocking fees, no lengthy interrogations.

---

## Return Eligibility

### Standard Window
Items may be returned within **30 days of the confirmed delivery date**.
Platinum Rewards members receive an extended window of **45 days**.

### Item Condition Requirements
To be accepted, returned items must be:
- Unworn, unwashed, and unaltered
- In original packaging with all tags attached
- Free from perfume, deodorant, pet hair, or visible wear

### Non-Returnable Items
The following cannot be returned under any circumstances:
- Underwear, swimwear, and pierced jewellery (hygiene policy)
- Items marked **Final Sale** at time of purchase
- Digital gift cards and store credit
- Items damaged by the customer after delivery

### Partial Returns
You may return individual items from a multi-item order. Each item is assessed
independently for eligibility.

---

## How to Initiate a Return

1. Contact support via chat or email within the return window
2. Provide your order ID and the item(s) you wish to return
3. State your reason from the accepted list (see below)
4. Receive a Return Merchandise Authorisation (RMA) number
5. Ship the item back within **7 days** of receiving your RMA
6. Leafy covers return shipping for defective/wrong items. Customer covers
   return shipping for change-of-mind returns.

### Accepted Return Reasons
These map directly to what the support agent can process:

| Reason | Return Covered By |
|---|---|
| Defective / damaged | Leafy (free return label) |
| Wrong item received | Leafy (free return label) |
| Item not as described | Leafy (free return label) |
| Changed mind | Customer pays return shipping |
| Size / fit issue | Customer pays return shipping |

---

## Refund Options

Once the returned item is received and inspected (1–2 business days), the customer
chooses one of three refund methods:

### 1. Original Payment Method
Refund to the card or payment method used at checkout.
- Processing time: 5–7 business days after item receipt
- Amount: Full item price (minus original shipping if change-of-mind return)

### 2. Store Credit
Issued instantly to your Leafy account upon return approval.
- Bonus: **+10% added** on top of item value (e.g. $50 item → $55 credit)
- No expiry on store credit
- Can be combined with discount codes

### 3. Bank Transfer
Available for customers without a saved payment method on file.
- Processing time: 7–10 business days
- Requires verified bank details on account

---

## Return Status Flow

Returns move through the following statuses (visible in your account and to the agent):

| Status | Meaning |
|---|---|
| Requested | Return initiated, awaiting approval |
| Approved | RMA issued, waiting for item to be shipped back |
| Item Received | Item arrived at Leafy warehouse, under inspection |
| Refund Issued | Refund processed and on its way |
| Rejected | Return denied (see rejection reasons below) |

### Common Rejection Reasons
- Item returned outside the return window
- Item arrived damaged by customer (not original defect)
- Item missing tags or original packaging
- Non-returnable item category

---

## Exchanges

Leafy does not process direct exchanges. To get a different size or colour:
1. Initiate a return for the original item
2. Place a new order for the correct item
3. Store credit refund is recommended to speed up the process

---

## Damaged or Defective Items

If your item arrives damaged or is defective:
- Report within **72 hours of delivery** for priority handling
- Provide photo evidence via the chat agent or email
- Leafy will offer: full refund, store credit, or replacement (subject to stock)
- No need to return the item in most defective cases — agent discretion applies
