# Leafy Order Cancellation Policy

## Cancellation Window

Orders can be cancelled **only before they enter the "Shipped" status**.
Once a shipping label is generated, the order cannot be cancelled — it must
go through the standard return process after delivery.

| Order Status | Can Cancel? | How |
|---|---|---|
| Processing | Yes | Via support chat or email immediately |
| Shipped | No | Wait for delivery, then initiate return |
| Delivered | No | Initiate return within return window |
| Cancelled | N/A | Already cancelled |

---

## How to Cancel

1. Contact support immediately with your order ID
2. Agent checks current order status in real time
3. If still in "Processing": cancellation is confirmed, refund initiated
4. If already "Shipped": agent explains return process and provides return policy

**Speed matters.** Orders move from Processing to Shipped quickly — often within
hours of placement on business days. Contact support as soon as possible.

---

## Refund on Cancellation

- Cancellations before shipment receive a **full refund** including original shipping cost
- Refund is issued to the original payment method
- Processing time: 3–5 business days to appear on statement
- Alternatively, store credit can be issued instantly (no bonus for cancellations,
  unlike returns)

---

## Partial Cancellations

If an order contains multiple items, individual items can be cancelled if the
order has not yet been packed. Once packing begins, partial cancellation is
not possible — the customer must wait and return the unwanted item.

---

## Cancelled by Leafy

Leafy may cancel an order in the following situations:

| Reason | Customer Action |
|---|---|
| Payment failed after 24-hour hold | Retry payment or reorder |
| Item went out of stock after order | Full refund issued automatically |
| Suspected fraud on the order | Contact support to verify identity |
| Address undeliverable | Update address and reorder |
| Customer in restricted region | Full refund issued |

In all Leafy-initiated cancellations, the customer receives a full refund to
the original payment method within 3–5 business days.

---

## Pre-orders & Backorders

- Pre-ordered items can be cancelled at any time before the item ships
- Backordered items follow standard cancellation rules from the point the item ships
- Customers are notified by email when a backordered item ships, with a 2-hour
  grace period to cancel before the cancellation window closes

---

## Subscription & Recurring Orders

Leafy does not currently offer subscription or auto-replenishment orders.
All orders are one-time purchases.

---

## Agent Reference: Cancellation Decision Tree

```
Customer wants to cancel order X
  └── Check order status
        ├── Status = "Processing"
        │     └── Confirm cancellation → issue full refund → done
        ├── Status = "Shipped" or later
        │     └── Explain: cannot cancel → walk through return process
        └── Status = "Cancelled"
              └── Confirm already cancelled → check if refund was issued
```

---

*Cancellation requests are processed in the order they are received.
Leafy cannot guarantee cancellation success for orders in active processing.*
